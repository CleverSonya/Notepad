Welcome to Notepad v 1.2 beta !
Author: PopovaSS

System requirements:
JVM down to 1.8.0. 

How to:
Notepad is portable.
Run a Notepad.exe based in the root directory or a NoteFrame.java based in /src/src/.
To upload a new custom font place a .ttf file in /user_data/fonts/ and restart the application.

Available features:
- multiple tabs
- basic font edition features: font family, size, color

Features coming soon:
- more style edition features with the ability to save style changes properly
